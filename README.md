# Shop page
>Catalog page of the online store. Developed with using SASS and Jade.

## [Go to the site](http://korchenov.ru/shop)
## Installation
>Required: Ruby

```sh
mkdir site
git clone https://github.com:KorilSalsen/shop-page.git site
cd site
gem install sass
gem install compass
gem install sass-globbing
npm install
bower install
gulp
```
